ผลิตภัณฑ์: PS-System

การเก็บประวัติพนักงานและการทำเงินเดือน

หากคุณกำลังมองหาซอฟต์แวร์สำหรับการทำเงินเดือนและการจัดเก็บข้อมูลพนักงาน

เราขอนำเสนอซอฟต์แวร์ PS-System

"เรียบง่ายและเต็มเปี่ยมด้วยประสิทธิภาพ"

จุดเด่นของผลิตภัณฑ์ และการบริการ

1. ผลิตภัณฑ์ออกแบบตามความต้องการของงานด้านบุคคล

2. ให้ความสำคัญกับความเรียบง่าย และมีประโยชน์ในการใช้งาน

3. ทำงานผ่านระบบ Cloud สามารถทำงานได้ทุกที่ ทุกเวลา ตามที่คุณต้องการ

4. ไม่มีขั้นตอนติดตั้งโปรแกรมให้ยุ่งยากสำหรับลูกค้า

5. ลดความยุ่งยากซับซ้อนสำหรับลูกค้าในการดูแลรักษาระบบแม่ข่าย (Server)

6. เรารับฟังความคิดเห็นของลูกค้าเพื่อพัฒนาผลิตภัณฑ์ เพื่อตอบโจทย์การทำงานด้านบุคคลอย่างสมบูรณ์แบบ

7. การเข้าใจความต้องการของลูกค้า และเข้าใจองค์กรเราเอง เราเชื่อมั่นว่าจะนำมาซึ่งความสมดุลในการบริการ


Organization Module

สามารถเก็บข้อมูลพื้นฐานขององค์กร ประกอบด้วย ชื่อ, ที่อยู่ ตำแหน่งงาน และหน่วยงานของบริษัท

Personnel Module

สามารถเก็บข้อมูลพนักงาน ประกอบด้วย ข้อมูลทั่่วไป, ข้อมูลการจ้างงาน, รายได้รายหักที่ได้รับทุกเดือน

การคำนวณกองทุนสำรองเลี้ยงชีพ, การศึกษา และประวัติการทำงาน

และรวมถึงการปรับตำแหน่ง เงินเดือน และการบันทึกข้อมูลพนักงานลาออกด้วย

Payroll Module

สามารถคำนวณการบันทึกจ่ายรายได้รายหักที่ไม่แน่นอนในแต่ละเดือน

สามารถคำนวณเงินเดือน รายได้รายหักที่ได้รับทุกเดือน และรายได้รายหักที่ได้รับไม่แน่นอนในแต่ละเดือน

เพื่อทำสรุปการจ่ายเงินเดือนให้กับผู้บริหาร และดำเนินการจ่ายเงินเดือนให้กับพนักงานต่อไป

Report Module

สามารถเรียกรายงานประวัติพนักงาน, รายงานสรุปเงินเดือน, รายได้ และรายหักของพนักงาน รวมถึงใบรับเงินเดือน

สามารถเรียกรายงานค่ากำหนดพื้นฐาน เช่น ชื่อองค์กร, ตำแหน่ง, หน่วยงาน และรายได้รายหัก เป็นต้น

Setup Module

สามารถกำหนดค่าพื้นฐานในส่วนของ ธนาคาร, กองทุนสำรองเลี้ยงชีพ, รายได้รายหัก

กลุ่มพนักงาน, ระดับพนักงาน และช่วงเวลาในการจ่ายเงินเดือน
